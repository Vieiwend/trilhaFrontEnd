Respostas

a) CSS Grid Layout introduz um sistema bi-dimensional de grid (literalmente "grades") para CSS. Grids podem ser usados para o design de layouts de grandes seções de uma webpage, assim como de pequenos elementos de interface.

b) Display-grid: Torna o elemento um grid container;

Display-inline-grid: Torna o elemento um grid container porém com comportamento inline;

Display-subgrid: Para grids dentro de grids (ainda não é suportado, porém você pode normalmente colocar display: grid; no grid dentro do grid que funciona);

Grid-template-columns: Define o número total de colunas que serão criadas no grid.
Ex: Grid-template-columns: 100px 100px 100px 100px; = Quatro colunas de 100px de largura são criadas.

Grid-template-rows: Define a quantidade de linhas no grid. Ex: grid-template-rows: 50px 100px 50px 150px;

Grid-template-areas: Define áreas específicas no grid. O ponto (.) pode ser utilizado para criar áreas vazias. Ex:grid-template-areas: "logo nav nav" = Cria 3 colunas e 3 linhas.

Grid-template: Atalho para definir o grid-template-columns, grid-template-rows e grid-template-areas.ex: grid-template: 50px 1fr / 100px 1fr 100px;

Gap: Define o gap (gutter) entre os elementos do grid. Ex: gap: 20px = Define 20px entre os elementos do grid (linha e coluna).

Grid-auto-columns: Define o tamanho das colunas do grid implícito (gerado automaticamente, quando algum elemento é posicionado em uma coluna que não foi definida). Ex: grid-auto-columns: 100px = As colunas implícitas, geradas automaticamente, terão 100px de largura.

Grid-auto-rows: Define o tamanho das linhas do grid implícito (gerado automaticamente, quando algum elemento é posicionado em uma linha que não foi definida). Ex: grid-auto-rows: 100px = As linhas implícitas, geradas automaticamente, terão 100px de altura.

Grid-auto-flow: Define o fluxo dos itens no grid. Se eles vão automaticamente gerar novas linhas ou colunas. ex: grid-auto-flow: row = Automaticamente gera novas linhas.

Grid: Atalho geral para definir o grid: grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns e grid-auto-flow. Ex: grid: 100px / 1fr 1fr = Gera uma linha com 100px de altura e 2 colunas com 1fr.

Justify-content: Justifica os itens do grid em relação ao eixo x (horizontal). Ex: justify-content: start = Justifica os itens ao início.

Align-content: Alinha os itens do grid em relação ao eixo y (vertical).ex: align-content: start = Alinha os itens ao início.

Justify-items: Justifica o conteúdo dos itens do grid em relação ao eixo x (horizontal). Justifica em relação a célula. Ex: justify-items: start// Justifica os itens ao início.

Align-items: Alinha o conteúdo dos itens do grid em relação ao eixo y (vertical). Alinha em relação a célula. Ex: align-items: start = Alinha os itens ao início.

c) Grid-item: Os Grid Itens são os filhos diretos do Grid Container. Um grid item pode ser explicito ou implícito. Explicito é quanto você define ele explicitamente no container e implícito é quanto ele é criado automaticamente para preencher o conteúdo no grid.

Grid-column: Define quais colunas serão ocupadas pelo grid item. É possível definir uma linha de início e final, assim o item irá ocupar múltiplas colunas. Ex: Grid-column: 1 = O item ocupará a coluna 1.

Grid-row : Define quais linhas serão ocupadas pelo grid item. Ex: grid-row: 1 = O item ocupará a linha 1.

Grid-area: Define a área do item do grid. É um atalho para grid-row-start,grid-column-start, grid-row-end, grid-column-end. Ex: grid-area: 1 / 2 / 4 / 3; = Este é um atalho para: grid-row-start: 1;grid-column-start: 2;grid-row-end: 4;grid-column-end: 3;

Justify-self: Justifica o item do grid em relação ao eixo x (horizontal). Justifica em relação a célula. Ex: justify-self: start// Justifica o item ao início.

Align-self: Justifica o item do grid em relação ao eixo y (vertical). Alinha em relação a célula. Ex: align-self: start = Alinha o item ao início.

d) FR é uma unidade fracional. Ex: grid-template-columns: 1fr 2fr; = Duas colunas são criadas, sendo a segunda com o dobro do tamanho da primeira. O tamanho do conteúdo é respeitado, ou seja, se o conteúdo na primeira coluna for maior que o da segunda, a primeira será maior.

e) Ex: repeat(3, 1fr); = Cria 3 colunas com 1fr de tamanho. O repeat seria a mesma coisa que escrever 1fr 1fr 1fr.

f) São as linhas que formam o grid. Diferentes das rows, que traduzindo também significam linhas, porém são propriedades distintas. Por exemplo, uma row possuí duas linhas, uma no inicio e uma no final.

g) É a menor unidade que você pode ter em sua grade CSS. É o espaço entre quatro linhas de grade que se cruzam e conceitualmente muito parecido com uma célula de tabela.

h) É uma ou mais células que compõem uma área retangular do grid. As Áreas grid quando se coloca um item usando line-based placement (posicionamento baseado em colunas e linha)ou quando define áreas usando named grid areas(substituindo o nome).
